<button type="button" data-bs-target="#certificatesCarousel" data-bs-slide-to="X" aria-label="Slide X+1"></button>

Steps to add a new certificate:

  1. Save the certificate image in assets/images/certificates/
  2. Copy the appropriate template above
  3. Replace the placeholders:
    • X - Next number in sequence
    • YOUR-CERTIFICATE-ID - The actual certificate ID from the URL
    • Course title and description
  4. Add the new carousel-item to the carousel-inner
  5. Add a new indicator button
  6. Update the slide-to numbers in the indicators
